Introduction

This sorting task requires you to distribute nine scores (-2, -1, -1, 0, 0, 0, +1, +1, and +2) across nine statements, in a bell curve.

  • Only one statement can receive a +2 and only one statement can receive a -2.
  • Two statements receive -2 and +2 scores respectively.
  • Three statements must receive a neutral score of 0.

Once you complete your sort, please click on the validate button to validate the result.

  • If the result is valid, a submit button will appear. Click on it will submit your sort result.
  • If the result is invalid, please go back to allocate correct numbers of statements to each score.

Click on the reset button will reset the ranking table.
